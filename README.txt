MIKE CHIMHUKA PORTFOLIO V2

How to use:
1. Put your professional photo inside the images folder.
2. Rename it exactly: profile.jpg
3. Upload index.html, style.css, script.js, assets, and images to your GitHub repository.
4. Commit changes.
5. Wait 1-5 minutes and refresh your GitHub Pages link.

Important:
- Update the LinkedIn link in index.html.
- Update the GitHub link if your username changes.
- Replace the resume file in assets when you have a final PDF resume.
